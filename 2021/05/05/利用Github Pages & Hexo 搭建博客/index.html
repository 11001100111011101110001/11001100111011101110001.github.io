利用Github Pages & Hexo 搭建博客

  • Github Pages：https://pages.github.com/
  • Hexo：https://hexo.io/
  • melody主题：https://github.com/Molunerfinn/hexo-theme-melody
  • stellar主题：https://github.com/xaoxuu/hexo-theme-stellar

Github Pages

GitHub Pages is designed to host your personal, organization, or project pages from a GitHub repository.

GitHub Pages是一种静态站点托管服务，旨在直接从GitHub仓库托管个人、组织或项目页面。

注册Github账号

  • 访问https://github.com/join创建GitHub账号。

创建Repository

  • 登录后访问https://github.com/new创建新的Repository。
  • Repository name需要以username.github.io命名，其中username为GitHub用户名。
  • 勾选Add a REAMD file初始化Repository。
  • 点击Create repository创建Repository。

初始化站点

  • 此时访问https://11001100111011101110001.github.io/即可看到hello world示例。

部署Hexo

快速、简洁且高效的博客框架。

将博客部署在 Github 上

部署主题

  • Hexo主题：https://hexo.io/themes/
  • melody主题文档：https://molunerfinn.com/hexo-theme-melody-doc/zh-Hans/
  • stellar主题文档：https://xaoxuu.com/wiki/stellar
  • 后期升级主题：npm update hexo-theme-melody

绑定域名

  • 添加类型为CNAME的记录，记录值为11001100111011101110001.github.io，其他默认。
  • 进入Repository，点击setting->Pages设置Custom domain为需要绑定的域名。
  • 进入Repository，点击setting->Pages勾选Enforce HTTPS。
  • 在source文件夹中新建CNAME文件，内容为blog.fuwenqing.com。
  • 配置_config.yml，将url配置项变更为https://blog.fuwenqing.com。
  • 重新部署hexo g -d。
